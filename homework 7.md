# рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 7: рЃЮрЃЌрЃ«рЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ + рЃЉрЃЮрЃюрЃБрЃА рЃЕрЃћрЃџрЃћрЃюрЃ»рЃў ­Ъџђ

рЃљрЃЏ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљрЃерЃў рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃўрЃљ рЃЮрЃЌрЃ«рЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ (рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў 0.5 рЃЦрЃБрЃџрЃўрЃЌ) рЃЊрЃљ рЃћрЃарЃЌрЃў рЃЉрЃЮрЃюрЃБрЃА рЃЕрЃћрЃџрЃћрЃюрЃ»рЃў (рЃљрЃа рЃерЃћрЃцрЃљрЃАрЃЊрЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃљрЃАрЃўрЃљрЃЏрЃЮрЃЋрЃюрЃЮрЃљ рЃљрЃЏрЃЮрЃ«рЃАрЃюрЃљ).

---

## 1. рЃЎрЃўрЃюрЃЮрЃЌрЃћрЃљрЃбрЃарЃерЃў рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃљ

### рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ:
рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃАрЃЎрЃарЃўрЃърЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃЌ:
```javascript
prompt("рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃгрЃџрЃўрЃА рЃ«рЃљрЃа?");
```

- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА рЃарЃўрЃфрЃ«рЃЋрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃюрЃљрЃЎрЃџрЃћрЃЉрЃўрЃљ 12-рЃќрЃћ, рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ:
  ```
  "рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃљ."
  ```
- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА рЃарЃўрЃфрЃ«рЃЋрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃћрЃбрЃўрЃљ рЃљрЃю рЃбрЃЮрЃџрЃўрЃљ 12-рЃўрЃАрЃљ, рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ:
  ```
  "рЃЌрЃЦрЃЋрЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃћрЃАрЃгрЃарЃЮрЃЌ рЃцрЃўрЃџрЃЏрЃА."
  ```

---

## 2. рЃерЃБрЃЦрЃюрЃўрЃерЃљрЃюрЃў

### рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ:
рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃЎрЃЮрЃЊрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃЌ:
```javascript
prompt("рЃарЃЮрЃЏрЃћрЃџрЃў рЃцрЃћрЃарЃўрЃљ рЃерЃБрЃЦрЃюрЃўрЃерЃљрЃюрЃќрЃћ?");
```

рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃцрЃћрЃарЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (рЃДрЃЋрЃўрЃЌрЃћрЃџрЃў, рЃгрЃўрЃЌрЃћрЃџрЃў, рЃЏрЃгрЃЋрЃљрЃюрЃћ), рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ:
- **рЃДрЃЋрЃўрЃЌрЃћрЃџрЃў:**
  ```
  "рЃЏрЃЮрЃћрЃЏрЃќрЃљрЃЊрЃћ"
  ```
- **рЃгрЃўрЃЌрЃћрЃџрЃў:**
  ```
  "рЃњрЃљрЃЕрЃћрЃарЃЊрЃў"
  ```
- **рЃЏрЃгрЃЋрЃљрЃюрЃћ:**
  ```
  "рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃгрЃљрЃ«рЃЋрЃўрЃЊрЃћ"
  ```

---

## 3. рЃЏрЃўрЃАрЃљрЃџрЃЏрЃћрЃЉрЃљ

### рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ:
рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃЎрЃЮрЃЊрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃЌ:
```javascript
prompt("рЃарЃЮрЃЏрЃћрЃџрЃў рЃАрЃљрЃљрЃЌрЃўрЃљ?");
```

- рЃЌрЃБ рЃЊрЃарЃЮ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃўрЃљ 12-рЃќрЃћ, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ:
  ```
  "рЃЊрЃўрЃџрЃљ рЃЏрЃерЃЋрЃўрЃЊрЃЮрЃЉрЃўрЃА"
  ```
- рЃЌрЃБ рЃЊрЃарЃЮ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃўрЃљ 18-рЃќрЃћ, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ:
  ```
  "рЃерЃБрЃљрЃЊрЃдрЃћ рЃЏрЃерЃЋрЃўрЃЊрЃЮрЃЉрЃўрЃА"
  ```
- рЃАрЃ«рЃЋрЃљ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ:
  ```
  "рЃАрЃљрЃдрЃљрЃЏрЃЮ рЃЏрЃерЃЋрЃўрЃЊрЃЮрЃЉрЃўрЃА"
  ```

---

## 4. рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

### рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ:
рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃАрЃЎрЃарЃўрЃърЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃЌ:
```javascript
prompt("рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЦрЃБрЃџрЃљ");
```

- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА 90 рЃљрЃю рЃЏрЃћрЃб рЃЦрЃБрЃџрЃљрЃА, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ:
  ```
  "A"
  ```
- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА 70 рЃљрЃю рЃЏрЃћрЃб рЃЦрЃБрЃџрЃљрЃА, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ:
  ```
  "B"
  ```
- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА 50 рЃљрЃю рЃЏрЃћрЃб рЃЦрЃБрЃџрЃљрЃА, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ:
  ```
  "C"
  ```
- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА 50-рЃќрЃћ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃА, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ:
  ```
  "F"
  ```

## рЃЉрЃЮрЃюрЃБрЃА рЃЕрЃћрЃџрЃћрЃюрЃ»рЃў ­Ъџђ

### рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ:
рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃАрЃЎрЃарЃўрЃърЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃЎрЃўрЃюрЃЮрЃЌрЃћрЃљрЃбрЃарЃўрЃА рЃЉрЃўрЃџрЃћрЃЌрЃўрЃА рЃцрЃљрЃАрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃАрЃљрЃЏрЃў рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ:

1. **рЃљрЃАрЃљрЃЎрЃў:**
   ```javascript
   prompt("Enter your age:");
   ```
2. **рЃцрЃўрЃџрЃЏрЃўрЃА рЃбрЃўрЃърЃў (рЃарЃћрЃњрЃБрЃџрЃљрЃарЃБрЃџрЃў рЃЌрЃБ 3D):**
   ```javascript
   prompt("Enter movie type (regular or 3D):");
   ```
3. **рЃЊрЃдрЃўрЃА рЃЊрЃарЃЮ (рЃЏрЃљрЃбрЃўрЃюрЃћ рЃЌрЃБ рЃАрЃљрЃдрЃљрЃЏрЃЮ):**
   ```javascript
   prompt("Enter time of day (matinee or evening):");
   ```

### рЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ:
- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃарЃўрЃА **10 рЃгрЃџрЃўрЃА рЃљрЃю рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў**, рЃЉрЃўрЃџрЃћрЃЌрЃўрЃА рЃцрЃљрЃАрЃў рЃљрЃарЃўрЃА **5 рЃџрЃљрЃарЃў**.
- рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃарЃўрЃА **65 рЃгрЃџрЃўрЃА рЃљрЃю рЃЏрЃћрЃбрЃў**, рЃЉрЃўрЃџрЃћрЃЌрЃўрЃА рЃцрЃљрЃАрЃў рЃљрЃарЃўрЃА **10 рЃџрЃљрЃарЃў**.
- рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃЉрЃўрЃџрЃћрЃЌрЃўрЃА рЃцрЃљрЃАрЃў рЃљрЃарЃўрЃА **15 рЃџрЃљрЃарЃў**.

#### рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃњрЃљрЃЊрЃљрЃАрЃљрЃ«рЃљрЃЊрЃў:
- рЃЌрЃБ рЃцрЃўрЃџрЃЏрЃўрЃА рЃбрЃўрЃърЃў рЃљрЃарЃўрЃА **3D**, рЃцрЃљрЃАрЃА рЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ **5 рЃџрЃљрЃарЃў**.
- рЃЌрЃБ рЃЊрЃдрЃўрЃА рЃЊрЃарЃЮ рЃљрЃарЃўрЃА **рЃАрЃљрЃдрЃљрЃЏрЃЮ**, рЃцрЃљрЃАрЃА рЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ **5 рЃџрЃљрЃарЃў**.

### рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃерЃћрЃЊрЃћрЃњрЃў:
рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЌ рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃцрЃљрЃАрЃў рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ:
```
"рЃЉрЃўрЃџрЃћрЃЌрЃўрЃА рЃцрЃљрЃАрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃљрЃарЃўрЃА: XX рЃџрЃљрЃарЃў."
```

рЃЌрЃЦрЃЋрЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃљ рЃљрЃю рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа рЃЏрЃўрЃАрЃў рЃерЃћрЃфрЃЋрЃџрЃљ!
рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃЏрЃЮрЃЏрЃгрЃћрЃарЃћрЃЌ! ­Ъџђ 
рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃћрЃЉрЃў РЮц№ИЈ
